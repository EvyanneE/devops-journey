# Docker and CI/CD Lab

Applied concepts from a beginner Docker tutorial to containerise a MERN stack web application.

## Concepts Practiced

- Built and containerised a simple web app
- Created a custom image using a Dockerfile
- Ran containers with port mapping
- Used volumes for data persistence

## Overall CI/CD Workflow

- GitHub Actions pipeline: push → build Docker image → push to registry → deploy container

## Skills Demonstrated

- Docker: images, containers, ports, volumes, Dockerfile
- Container registry usage (to push/pull images)
- GitHub Actions: CI/CD automation
