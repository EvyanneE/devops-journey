# Bash Scripting Fundamentals

Applied concepts from a Bash scripting tutorial to build practical automation scripts.

## Concepts Practiced

- Variables
- Conditionals
- Loops
- Functions
- File Operations

## Files Included

- `analyse-logs.sh` – Loops through all log files updates in the last 24 hour and saves to the file `log-analysis-report` the errors in those log files and the count of each error type. If the count is above a set limit, a warning is printed to the terminal.
- `logs/` – Log files provided by the tutorial and log analysis output from `analyse-logs.sh`

## Outcome

Developed foundational shell automation skills essential for DevOps workflows.
